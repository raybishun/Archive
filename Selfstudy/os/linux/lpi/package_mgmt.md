# Package Management

## RPM
```
dnf check-update (if fedora)
yum check-update (if centOS)

sudo yum upgrade

yum search google
sudo yum install chrome-remote-desktop.x86_64
sudo yum erase chrome-remote-desktop.x86_64
```

## DEB
```
sudo apt update (if ubuntu)

sudo apt upgrade

sudo apt autoremove (optional cleanup)

yum search google

apt show <package_name>

apt install <package_name>

sudo apt remove <package_name>
```
